# Event-Reservation-Management-Using-API

## Overview

A database-driven console application built using Core Java and MySQL that enables users to securely register, authenticate, browse events, and reserve seats.
The project demonstrates real-world backend concepts such as JDBC integration, data persistence, structured program flow, and transaction handling.

This application was developed to strengthen practical knowledge of Java backend development and relational database interaction.

## Tools

1. Programming Language: Java (Core Java)

2. Database: MySQL

3. Connectivity: JDBC
4.  SQL Queries
5.  Console-based UI Design

## How to Execute

1. Configure MySQL database using the provided schema

2. Update database credentials in DBConnection.java

3. Run MainMenu.java

4. Interact through the console to register, log in, and reserve events
